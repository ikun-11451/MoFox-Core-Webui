## 📋 概述

MoFox WebUI 初始化系统为首次登录的用户提供友好、直观的配置向导，帮助用户快速完成机器人的基础配置。

### 核心功能

- 🤖 **机器人配置** - 昵称、人格、身份、回复风格等
- 🧠 **AI 模型配置** - SiliconFlow API 快速接入
- 🔧 **Git 路径配置** - 系统更新功能支持

### 核心特性

- ✨ **智能检测** - 自动检测已有配置，避免重复设置
- 🎯 **渐进式引导** - 分步骤完成配置，降低认知负担
- 🎨 **视觉愉悦** - Material Design 3 风格，动画流畅
- 🎁 **趣味彩蛋** - 根据日期和节日显示不同的问候语和装饰
- 💾 **安全可靠** - 配置前自动备份，支持跳过已配置项

## 🎯 设计目标

### 功能目标

1. **降低上手难度** - 新用户无需查阅文档即可完成基础配置
2. **避免重复配置** - 检测已有配置，提示用户并允许跳过
3. **配置验证** - 实时验证用户输入，提供友好的错误提示
4. **最小化配置** - 仅配置必要项，引导用户后续在编辑器中完成高级配置

### 体验目标

1. **温馨友好** - 通过问候语和装饰营造温馨氛围
2. **信息透明** - 清晰说明每个配置项的作用和影响
3. **操作简便** - 支持键盘导航、表单自动完成
4. **视觉引导** - 使用进度指示器、动画引导用户视线

## 🚀 快速开始

### 开发流程

1. **阅读设计文档** - 从 [初始化流程设计](./flow-design.md) 开始
2. **实现后端 API** - 参考 [后端 API 设计](./api-design.md)
3. **开发前端组件** - 遵循 [前端实现指南](./frontend-guide.md)
4. **添加 UI 细节** - 根据 [UI 界面设计](./ui-design.md) 完善界面
5. **集成彩蛋功能** - 最后添加 [小彩蛋](./easter-eggs.md) 提升趣味性

### 实现优先级

#### P0（必须）
- ✅ 初始化状态检测
- ✅ 三个配置步骤的 UI 和逻辑
- ✅ 配置保存和验证
- ✅ 初始化完成标记

#### P1（重要）
- 🎯 欢迎页面和完成页面
- 🎯 基础问候语系统（时间段）
- 🎯 跳过已配置项的提示
- 🎯 错误处理和重试

#### P2（优化）
- 🎁 节日问候语和装饰
- 🎁 背景粒子动画
- 🎁 配置测试功能
- 🎁 Git 自动检测

#### P3（彩蛋）
- 🎉 隐藏彩蛋互动
- 🎉 随机趣味问候
- 🎉 Logo 动态效果
- 🎉 完成页小贴士

## 🛠️ 技术栈

- **前端框架**: Vue 3 + TypeScript
- **状态管理**: Pinia
- **路由**: Vue Router
- **UI 风格**: Material Design 3
- **动画**: Vue Transition + CSS Animations
- **图标**: Material Symbols
