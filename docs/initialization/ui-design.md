# UI 界面设计

> 本文档详细描述初始化系统各个界面的布局、交互和视觉设计

---

## 🎨 设计原则

### Material Design 3

- **动态色彩系统** - 使用 MD3 动态颜色
- **圆角设计** - 统一使用 28px 圆角
- **高度系统** - 使用 elevation 表示层级
- **状态层** - 悬停、按下、聚焦状态

### 布局原则

- **居中对齐** - 主要内容居中显示
- **最大宽度** - 卡片最大宽度 720px
- **间距规范** - 使用 8px 基准间距
- **响应式** - 适配桌面、平板、移动端
